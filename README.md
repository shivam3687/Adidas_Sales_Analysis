# Adidas Sales Analysis

## Project Overview

This project analyzes Adidas sales data to uncover key insights about product performance, regional sales trends, and customer purchasing behavior.
Using Python and data visualization libraries, the project explores how different factors such as region, product category, and sales method influence overall revenue and profitability.

The goal is to transform raw sales data into meaningful insights that can help businesses make better strategic decisions.

---

## Dataset Description

The dataset contains information about Adidas product sales including:

* Retailer
* Invoice Date
* Region
* State
* City
* Product Category
* Price per Unit
* Units Sold
* Total Sales
* Operating Profit
* Operating Margin
* Sales Method

These features allow us to analyze sales performance across multiple dimensions such as geography, product category, and sales channels. ([GitHub][1])

## Exploratory Data Analysis (EDA)

The following analyses were performed:

* Regional sales distribution
* Product category performance
* Sales trends across different states
* Profit and margin analysis
* Sales method comparison

Data visualization techniques were used to better understand patterns and trends in the dataset.

---

## Key Insights

* Certain regions generate significantly higher sales compared to others.
* Footwear and apparel categories contribute the largest share of revenue.
* Online and in-store sales channels show different purchasing patterns.
* Some cities and states dominate Adidas product sales.

These insights can help companies optimize marketing strategies and inventory planning.
